# Milena & Daniel Travels

Prywatny planer podróży dla Mileny i Daniela.

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Wrzuć cały folder projektu na GitHuba.
2. Wejdź na Vercel.
3. Kliknij **Add New... > Project**.
4. Wybierz repozytorium z tym projektem.
5. Framework powinien wykryć się jako **Vite**.
6. Kliknij **Deploy**.
7. Po deployu otwórz link na telefonie.
8. W Safari/Chrome wybierz **Dodaj do ekranu głównego**.

## Uwaga

Ta wersja zapisuje dane lokalnie w przeglądarce. Kolejny krok to dodanie Firebase/Supabase, żeby dane synchronizowały się między Danielem i Mileną.
